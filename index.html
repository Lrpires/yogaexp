🔑 Configure o Token do GitHub

Para salvar os dados no GitHub, você precisa de um token de acesso pessoal. Ele fica armazenado somente neste navegador e nunca aparece no código público.

  1. Acesse github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens
  2. Clique em Generate new token
  3. Em Repository access, selecione "Only select repositories" e escolha yogaexp
  4. Em Permissions → Contents, selecione Read and write
