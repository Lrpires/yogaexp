🔗 Conectar ao Google Sheets

Para salvar todos os dados na nuvem e acessar de qualquer dispositivo, siga os passos abaixo para criar o backend no Google Apps Script.

  1. Abra Google Sheets, crie uma planilha nova e chame de Yoga Experience.
  2. No menu, clique em Extensões → Apps Script.
  3. Apague o código existente e cole todo o conteúdo do arquivo apps-script-backend.gs (que veio junto com este HTML).
  4. Clique em Implantar → Nova implantação → Tipo: App da Web → Executar como: Eu → Acesso: Qualquer pessoa → Implantar.
  5. Autorize o acesso quando solicitado, copie a URL da implantação e cole abaixo.
Pular por agora (dados ficam só neste navegador)
